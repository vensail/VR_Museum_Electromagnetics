# VR_Museum_Electromagnetics
It is a project made by unity. You can watch videos about Electromagnetics. For children, they can learn sth about Electromagnetics through this project. And it can also do sth to spread information of Electromagnetics.

First of all, users can creat an account whit their unique username and passcode. After that users could be allowed to start this program.
